# **App Name**: PhotoFlow

## Core Features:

- User Authentication: Secure user registration and login with JWT and bcrypt.
- Image Gallery: Display a responsive and visually appealing photo gallery with lazy loading.
- Image CRUD: Admin dashboard for creating, reading, updating, and deleting images.
- Shopping Cart: Allow users to add images to a cart and proceed to checkout.
- Stripe Integration: Process payments securely via Stripe for image purchases.
- Contact Section: A contact form that notifies the admin. LLM tool assists in determining if the contents are satisfactory
- Admin Dashboard: Manage site content (key-value pairs) via a JSON-based configuration.

## Style Guidelines:

- Primary color: Deep navy blue (#243A73) to convey trust and sophistication, aligning with the professional photography theme.
- Background color: Light gray (#F0F4F8) to ensure the images stand out without distraction, in line with the dark scheme.
- Accent color: Muted lavender (#A397D6) to provide highlights and call-to-action cues that feel both modern and calming.
- Headline font: 'Playfair', a modern serif with high contrast; body font: 'PT Sans' (sans-serif), for readability.
- Use modern, minimalist icons for navigation and user interactions.
- Implement a grid-based layout for the photo gallery to ensure responsiveness and visual consistency.
- Subtle, smooth animations using Framer Motion for transitions and interactions to enhance user experience.
